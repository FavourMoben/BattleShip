# How to Run/Play **Battle Ship** Game
-----------------------------------------
how to run the game
-------------------
**Type this on terminal**
```terminal

>cd Assigment1/
>java -jar assignment1-1.0-SNAPSHOT.jar width height numberOfships
```
then ...
--------
if you don't find the "assignment1-1.0-SNAPSHOT.jar" file, 
**Type this on terminal**

```terminal

>cd Assigment1/Favour_Jam
>mvn package
>cd target
>java -jar assignment1-1.0-SNAPSHOT.jar width height numberOfships
```
how to run the test 
-------------------
**Type this on terminal**
```terminal

>cd Assigment1/Favour_Jam 
>mvn clean test
```
